# Cabin-Resort
# 🌲 Cootels - Eco-Resort & Cabin Booking Landing Page

Саҳифаи аввали (Landing Page) муосир ва ҷозибадор барои меҳмонхонаи экологии **Cootels** (Cottage + Hotel), ки барои бандкунии котеҷҳо ва хонаҳои чӯбӣ дар батни табиат хизмат мекунад. Ин лоиҳа бо таваҷҷӯҳи махсус ба ҷузъиёт, тарҳи ҷавобгӯ (Responsive) ва усулҳои пешқадами веби муосир сохта шудааст.

---

## 🚀 Технологияҳои истифодашуда (Tech Stack)

Дар ин лоиҳа технологияҳои зерини касбӣ истифода шудаанд:
* **HTML5** - Барои сохтани сохтори дурусти семантикии саҳифа.
* **Tailwind CSS** - Фреймворки муосири Utility-first барои сохтани стилҳои тез, сабук ва аниматсияҳои мулоим.
* **Google Fonts & SVG Icons** - Барои визуализатсияи зебо ва истифодаи иконкаҳои векторӣ бидуни вазнин кардани сайт.

---

## ✨ Хусусиятҳои Лоиҳа (Features)
1. **Семантика ва Оптимизатсия:** Код мувофиқи стандартҳои SEO ва дастрасӣ (Accessibility) бо истифода аз тегҳои `header`, `main`, `section` ва `footer` сохта шудааст.
2. **Эффекти "Нардбон" ва Аккордеон:** Истифодаи классҳои пешрафтаи Tailwind барои сохтани галереяи динамикӣ дар қисми Hero ва блокҳои интерактивӣ барои қадамҳои бандкунии ҷой.
3. **Ислоҳи Имлоӣ ва Грамматика:** Ҳамаи хатогиҳои матнии тарҳи аввалия ҳангоми таҳияи код ба шакли касбии англисӣ ислоҳ карда шуданд.

---

## 📂 Сохтори Саҳифаҳои Лоиҳа

Лоиҳа аз бахшҳои (Sections) зерин иборат аст:
* **Header / Navigation:** Панели пайвандҳо ва тугмаи "Get Started".
* **Hero Section:** Сарлавҳаи ҷолиб, тавсиф ва галереяи сеқисмаи хонаҳо.
* **Cottage Info (Norway):** Муаррифии меҳмонхона бо расми амудии касбӣ.
* **Cabin Activities:** Бахши фаъолиятҳо ва хизматрасониҳо барои мизоҷон.
* **100% Organic Food:** Муаррифии бахши хӯроквории органикӣ бо расми мураббаъ.
* **Many Rooms to Choose:** Блоки 3-сутуна барои намоиши намудҳои хонаҳо (Single, Double, Cottage).
* **How to Get My Room:** Дастурамали зина ба зина (Steps UI) бо сояҳои мулоим.
* **Testimonials:** Бахши фикру мулоҳизаҳои мизоҷон бо аватари доиравӣ ва пагинация.
* **Footer:** Қисми поёнии сайт бо сохтори 5-сутуна ва ҷойгоҳи шабакаҳои иҷтимоӣ.
